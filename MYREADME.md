# DJS01: Vanilla JS Podcast App

## Description 📜:

We were tasked to create a landing page for a Podcast App. The page needed (to be responsive) to have desktop, tablet and mobile view. Usability was limited to certain amount of features, as this is a project we are to build upon - until it becomes a fully usable Podcast App, with full functionality.

## Task 🖊️:

- To use: OOP, SOLID, abstractions, modules and modular coding.
- To use: Vanilla JavaScript.

## Techonolgy 💻:

- HTML
- CSS
- JavaScript

## Surprises 👀 and Solutions 💡:

### Problem:
There were *no* image icons - to input into specific and necessary parts of the landing page.

### Solution:
I Googled, saved and used images from www.freepik.com for those missing key features of my Podcast landing page.

## Features 🎨:

- Podcast cards.
- Within the cards: Podcast - title, seasons, genre and last updated date.
- Modal: once having clicked on a Podcast card, you get more information like - description and an extended list of seasons with amount of episodes per season.
- Scroll bars: for both the Podcast landing page and modal.
- Button: to exit modal.
- Blur effect: once modal is opened.
- Responsiveness (changing screen viewing visuals per screen size): desktop, tablet, mobile.
